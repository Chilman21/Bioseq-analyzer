# 🧬 BioSeq Analyzer

A Python-based DNA sequence analysis toolkit built for biotechnology students.

## Features
- DNA validation
- Nucleotide counting
- GC content calculation
- Complement & Reverse Complement
- Transcription (DNA → RNA)
- Translation (DNA → Protein)
- Mutation detection
- Report generation

## Technologies Used
- Python
- Basic Bioinformatics logic

## How to Run
1. Download the repository
2. Run: python bioseq_analyzer.py
3. Enter DNA sequence when prompted
